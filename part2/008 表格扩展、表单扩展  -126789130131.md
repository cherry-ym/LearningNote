#### 表单扩展：

`新的input控件` 

   email : 电子邮件地址输入框

   url : 网址输入框

   number : 数值输入框

   range : 滑动条

   date / month / week  : 日期控件 

   search : 搜索框

   color : 颜色控件

   tel : 电话号码输入框（在移动端默认调出数字键盘）

   time : 时间控件

`表单的新属性`

  autocomplete : 自动完成		默认 on/off（关闭）

  autofocus : 获取焦点

  required : 不能为空

  pattern  : 正则验证

  与后端连接

  method :  数据传输方式

  enctype :  数据传输类型

  name / value : 数据的键值对
